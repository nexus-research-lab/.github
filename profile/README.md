# Nexus Research Lab

Open research, public code, and executable ideas.

We build in the open around agents, research infrastructure, knowledge systems, and human-AI workflows.

## Focus

- Agent systems and AI tooling
- Research infrastructure and developer tools
- Experimental interfaces and workflows
- Open-source prototypes, datasets, and lab notes

## How we work

- Build in public when possible
- Prefer simple, inspectable systems
- Share experiments early
- Treat repositories as research artifacts, not just code dumps
